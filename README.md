# General Knowledge Quiz 🎯
A simple Python-based quiz game that tests users with general knowledge questions and keeps track of their score. This beginner-friendly project demonstrates the use of variables, user input, conditional statements, and score tracking in Python.

# Features
- Asks multiple general knowledge questions.
- Validates user answers using if-else conditions.
- Awards points for correct answers.
- Displays the final score at the end of the quiz.
- Simple and interactive command-line interface.

#Technologies Used
- Python 3

# How It Works
1. The program asks the user a series of general knowledge questions.
2. The user enters their answers through the keyboard.
3. The program checks each answer and updates the score.
4. After all questions are answered, the final score is displayed.

# Learning Outcomes
This project helps beginners understand:
- Variables and data types
- User input handling
- Conditional statements (if-else)
- Basic program flow and logic
- Score tracking in Python
